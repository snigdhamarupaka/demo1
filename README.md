# React User Form App

A simple React application with a form to collect user information.

## Features

- Collects Name, Mobile Number, and Email
- Form validation
- Real-time form preview
- Success message on submission
- Responsive design

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Form Fields

- **Name**: Required field for full name
- **Mobile Number**: Required, must be 10 digits
- **Email**: Required, must be a valid email format
